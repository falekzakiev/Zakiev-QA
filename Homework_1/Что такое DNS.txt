Что такое DNS (Domain name System)  и в чем разница протоколов HTTP/HTTPS и их версий?

Domain name System переводится как Система доменных имен. Это распределенная система хранения, а также обработки информации о доменах. Структуру DNS можно представить в виде некого дерева, состоящего из распределенной базы имен. Его еще называют пространством имен домена. Так же благодаря DNS, IP-адреса сайтов, состоящие из множества числе, стали присваиваться их “доменные имена”, что облегчило конечным пользователям способ взаимодействия с сайтами. Ведь гораздо проще запомнить какое-то название, а не набор цифр.

HTTP - это открытый протокол. А HTTPS – соответственно закрытый. Те при использовании закрытого протокола вся передаваемая информация будет шифроваться сайтом. Что намного безопаснее, чем при открытом протоколе, где вся информация будет может передаваться третьим лицам.
Изначальная версия HTTP была версия 0, затем 0.9, за ней появилась версия 1.0, затем 1.1 и на данный момент это версия HTTP/2. Новый протокол решает наиболее существенную проблему передачи данных через интернет, а именно несколько открытых соединений. Те передача будет происходить в рамках одного соединения между клиентом и сервером. Что значительно ускорит и оптимизирует загрузку содержимого сайта.
